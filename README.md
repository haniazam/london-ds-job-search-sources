# London DS Job Search Sources

Repository for the London senior data scientist job-search project.

## Contents

- `docs/` project brief, handoff, and update protocol
- `prompts/` notebook rebuild prompt
- `data/raw/uploads/` text copies of source notes
- `data/exports/workbook/` CSV workbook exports
- `sources/` extracted job links and source index
- `resumes/` text versions of current and archived resume sources
- `scripts/` helper scripts

## Notes

This repository was rebuilt from the current project sources available in ChatGPT. Binary files are represented by text or markdown extractions where direct binary upload was not available through the connector.
